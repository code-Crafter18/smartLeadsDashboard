# Smart Leads Dashboard

A full-stack Lead Management Dashboard built using the MERN stack with TypeScript, JWT authentication, role-based access control, advanced filtering, pagination, Docker support, and MongoDB Atlas integration.

---

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Password Hashing using bcrypt

### Role-Based Access Control
#### Admin
- Create Leads
- View Leads
- Update Leads
- Delete Leads

#### Sales User
- Create Leads
- View Leads
- Update Leads

### Leads Management
- Create Lead
- Update Lead
- Delete Lead
- View Single Lead
- View Leads List

### Advanced Filtering & Search
- Filter by Status
- Filter by Source
- Search by Name or Email
- Sort by Latest / Oldest
- Debounced Search

### Pagination
- Backend Pagination
- Limit & Skip Implementation
- Pagination Metadata

### UI Features
- Responsive Dashboard
- Reusable Components
- Dark Mode
- Loading States
- Error Handling UI
- Confirmation Modals
- Clean Dashboard UI

### Additional Features
- CSV Export Functionality
- Docker Setup
- MongoDB Atlas Integration
